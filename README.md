# This is a mock restaurant website

### This site was built with React and vanilla CSS.

The purpose of the site is to display design and programming skills in a "real-world" application. 

Packages: 
  -React
  -React-Hook-Form
  -GSAP 3
  -Font Awesome
  -React Icons
 
Features: 
  -Menu overlay 
  -Modal Reservation form from various access points on page 
  -Animation
  -Flexbox
  -Grid
  -React Hooks
  -Mobile Responsiveness
  -Different designs/layouts for Desktop and Mobile views
